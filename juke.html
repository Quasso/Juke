Juke.js

A funky little responsive javascript jukebox that uses AJAX to make a feed of tracks from Last.fm, all playable via Spotify mobile or desktop

About

Thanks for checking this out. The reason I made this stems from me wanting to experiment with my portfolio site, and trying to think of creative ways to engage people and let them know a bit about me.

I originally wrote it using PHP, but I thought it would be a nice thing to share around, since people could use it for all kinds of things--so I re-wrote it fully in jQuery, and it uses Velocity.js for the animations. It currently only gels well with "user." methods which is largely due to the whacky Last.FM API.

To set options, you'll notice there's just an empty div with class "juke" on the page, the options you'll need for now are already inside the examples. Much love from Alex G.
